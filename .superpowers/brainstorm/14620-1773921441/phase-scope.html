What are we building first?

The plan has 4 phases. Let's agree on the starting scope.

A

Phase 1 — Core Game Only

Scaffold the Next.js app, build the <GameCard> and <Die> components, wire the game engine, build /play and the landing page. Free game works end-to-end. No auth, no Supabase, no Stripe yet.

B

Phase 1 + 2 — Game + Subscriptions

Everything in Phase 1, plus Supabase auth, Stripe checkout, pricing page, and paywall gates. Someone can pay €2.99 by the end.

C

Full Spec — All 4 Phases

Design the complete system end-to-end (game + auth + subscriptions + multiplayer rooms + dashboard) before writing any code. Longer upfront, but complete picture.
