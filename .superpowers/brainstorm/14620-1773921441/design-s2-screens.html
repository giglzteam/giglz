Screens & Game Flow
Every screen in the app, the game loop, and all page routes.

Game Loop (Single Phone)

1
Player Setup
Host opens /play. Adds 2–8 player names + picks emoji for each. Taps "Start Game". Deck is shuffled (75 cards, free tier shows first 25).
2–8 playersemoji pickerdeck shuffle
2
Roll Phase
Current player's name shown. Phone passed to them. They tap "Roll & Draw" — die animates, random 1–6 result. A card PNG is drawn from the shuffled deck. Mode strip shows challenge type + explanation.
die animationcard drawnmode strip updates
3
Reveal Phase
Full card PNG displayed. Mode strip shows active die number highlighted. 60-second timer bar starts. Player performs the challenge. Host (or any player) taps Correct ✓ or Skip →.
60s timercard PNG showncorrect / skip
4
Score & Advance
Correct → +1 point to current player. Score bar updates. Check if score ≥ 10 (cardsToWin). If yes → Winner screen. If no → next player, back to Roll Phase.
+1 score pop animationwin check
5
Winner Screen
Confetti animation. Winner name + emoji shown large. Final scores. Buttons: "Play Again" (reshuffle, same players) and "New Game" (back to setup).
confettifinal scoresplay again
